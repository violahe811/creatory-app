# Creator Product App

This project is a multi-screen mobile application built using React Native and Expo.

The app allows users to browse a list of products and view detailed information by tapping on a product. The navigation between screens is handled using React Navigation, including both a Stack Navigator and a Tab Navigator.

The Home tab contains a product list screen, where users can select an item to navigate to a detail screen. Data is passed between screens using navigation parameters and accessed using route.params.

The app also includes a Settings tab as a second main section, demonstrating tab-based navigation.

This project demonstrates key concepts such as:
- Stack navigation (list to detail)
- Tab navigation (Home and Settings)
- Passing data between screens
- Using FlatList to render data
- Configuring screen headers and tab icons

This simulates a simple product browsing experience similar to e-commerce or influencer product selection apps.
